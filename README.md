# Vietnamese-Sentiment-Assistant
## 1. Tải và giải nén thư mục
Tải project dưới dạng file .zip <br>
Giải nén ra một thư mục (ví dụ: Vietnamese-Sentiment-Assistant.zip) <br>
Mở PowerShell hoặc CMD, dùng lệnh sau để truy cập vào thư mục (đảm bảo đã tải 3.7 python version): <br>
    cd "Đường_dẫn_đến_thư_mục\Vietnamese-Sentiment-Assistant"
### 2. Tạo môi trường ảo
    py -m venv venv( vevv là tên môi trường )
### 3. Kích hoạt môi trường
    .\venv\Scripts\Activate
### 4. cài đặt các thư viện cần thiết
  	pip install -r requirements.txt
    pip install torch transformers underthesea 
### 5. sau khi cài xong chạy app
    python desktop_app.py
